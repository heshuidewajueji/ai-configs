# King AI Skills

个人AI Skills集合，支持Obsidian工具链、产品文档生成、学习助手等多种场景。

## 📦 Skills 索引

### 🛠️ 工具类

| Skill | 功能 | 安装 |
|-------|------|------|
| **defuddle** | 网页内容提取，移除广告和导航，节省token | `npx skills add heshuidewajueji/king-ai-skills/defuddle` |
| **skill-manage** | Skills自动分类与同步管理 | `npx skills add heshuidewajueji/king-ai-skills/skill-manage` |

### 📝 Obsidian 工具链

| Skill | 功能 | 安装 |
|-------|------|------|
| **obsidian-markdown** | Obsidian flavored markdown语法 | `npx skills add heshuidewajueji/king-ai-skills/obsidian-markdown` |
| **obsidian-cli** | Obsidian命令行交互 | `npx skills add heshuidewajueji/king-ai-skills/obsidian-cli` |
| **obsidian-bases** | Obsidian Bases数据库 | `npx skills add heshuidewajueji/king-ai-skills/obsidian-bases` |
| **json-canvas** | JSON Canvas可视化画布 | `npx skills add heshuidewajueji/king-ai-skills/json-canvas` |
| **tutor** | Obsidian学习Quiz助手 | `npx skills add heshuidewajueji/king-ai-skills/tutor` |

### 🚗 专业领域

| Skill | 功能 | 安装 |
|-------|------|------|
| **tbox-prd-skill** | 汽车TBOX产品需求文档 | `npx skills add heshuidewajueji/king-ai-skills/tbox-prd-skill` |

## 🔧 安装方式

### 方式一：按skill安装
```bash
# 单独安装某个skill
npx skills add heshuidewajueji/king-ai-skills/[skill-name]
```

### 方式二：克隆整个仓库
```bash
git clone https://github.com/heshuidewajueji/king-ai-skills.git ~/.hermes/skills/king-ai-skills
```

## 📂 目录结构
