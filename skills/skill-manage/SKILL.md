# Skill-Manage

> Skills 自动分类与同步管理

## 触发词

当 King 说「整理 Skills」或「同步 Skills」时使用。

## 功能

1. 扫描skills目录结构
2. 读取每个skill的SKILL.md
3. 按category/tag分类
4. 生成索引报告
5. 检测过时skills
